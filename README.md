# @mnrendra/lssync
To get all file informations in a directory

## Features
* lsSync

## Installation
```
npm i @mnrendra/lssync
```

## Example
```c
const lsSync = require('@mnrendra/lssync')
const allFiles = lsSync('~/')
console.log(allFiles)
```
